208 // socket. Trying to do scatter/gather IO with stream processing
209 // engine which might involve TLS/SSL does not make any sense
210 // as scatter/gather is not implemented, at least for OpenSSL.
211 // Therefore we write the user provided buffer into single
212 // continuous memory block which is passed to write function.

214 // async mode so the memory overhead should be minimal.
