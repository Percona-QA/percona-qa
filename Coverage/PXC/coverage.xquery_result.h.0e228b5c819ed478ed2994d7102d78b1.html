40 /**
41 Interface responsible for fetching 'resultsets'.
42
43 The interface defines methods that correspond to X Protocol message flow
44 which transmits multiple 'resultsets':
45
46 loop has more resultsets
47 group resultset
48 loop has more columns
49 server --> client: ColumnMetaData
50 end
51 loop has more rows
52 server --> client: Row
53 end
54 end
55 alt has more resultsets
56 server --> client: FetchDoneMoreResultsets
57 end
58 end
59 loop has more OUT-paramsets
60 server --> client: FetchDoneMoreOutParams
61 group resultset
62 loop has more columns
63 server --> client: ColumnMetaData
64 end
65 loop has more rows
66 server --> client: Row
67 end
68 end
69 end
70 server --> client: FetchDone
71
72 While designing it there was an assumption that the implementation may not
73 fetch/buffers row data. Because of it some return containers may "grow" with
74 the realization of the X Protocol flow.
75 */
