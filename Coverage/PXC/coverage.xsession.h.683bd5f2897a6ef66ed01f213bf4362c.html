50 /**
51 Capabilities supported by client library.
52
53 Capabilities are settings that are transfered between client and server
54 to change the behavior of both ends of X Protocol.
55 Capabilities listed here are handled by implementation of
56 XSession class. Setting capability may require to do some additional
57 reconfiguration of communication channel. For example setting TLS
58 capability reconfigures communication channel to use encryption.
59 When setting Capabilities manually (using XProtocol interface) user
60 needs to remember about the possible reconfiguration.
61 Capabilities must be set before connecting through XSession interface.

72 Capability type: BOOL. Value: enable/disable the support.

81 Capability_client_interactive,

84 The server may want know more about the client. MySQL X clients connection

87 about clients/statistics is going to help server administrators with
88 finding faulty clients, targeting potential issues and allow better

90 Capability type: OBJECT. Value: associative array of strings where
91 key name must not exceed 32 characters and value must not exceed
92 1024 characters.
93 */
94 Capability_session_connect_attrs,

97 Enable compression and choose the algorithm and style.

99 Capability type: OBJECT.
