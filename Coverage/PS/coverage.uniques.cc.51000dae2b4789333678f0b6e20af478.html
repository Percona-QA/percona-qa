24 /*
25 Function to handle quick removal of duplicates
26 This code is used when doing multi-table deletes to find the rows in
27 reference tables that needs to be deleted.
28
29 The basic idea is as follows:
30
31 Store first all strings in a binary tree, ignoring duplicates.
32 When the tree uses more memory than 'max_heap_table_size',
33 write the tree (in sorted order) out to disk and start with a new tree.
34 When all data has been generated, merge the trees (removing any found
35 duplicates).
36
37 The unique entries will be returned in sort order, to ensure that we do the
38 deletes in disk order.
39 */
